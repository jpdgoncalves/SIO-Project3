# SIO-Project2
Segundo Projeto de SIO

## How to run the code
After cloning the repository and creating a virtual environment with all the requirements:
- Open 2 consoles
- In one of them run:
> source venv/bin/activate
-
> python3 server.py
- In the other one run:
> source venv/bin/activate
-
> python3 client.py <name_of_file>
